DOFBOT-IK-Workspace
6축 로봇 팔(DOFBOT) 역운동학 제어 및 작업 공간 분석 라이브러리

본 저장소는 Yahboom DOFBOT의 실제 물리적 규격을 기반으로 한
역운동학(Inverse Kinematics) 계산, 하드웨어 보호를 위한 안전 영역 검증,
그리고 로봇의 가동 범위(Workspace) 분석 도구를 제공합니다.

📌 프로젝트 개요

이 라이브러리는 다음과 같은 목적을 갖습니다.

좌표 기반 로봇 제어를 위한 안정적인 IK 해 계산

실제 하드웨어 손상을 방지하기 위한 사전 안전 검증

로봇이 도달 가능한 물리적 작업 공간의 정량적 분석

로봇 제어 실습, 자동화 프로젝트, 연구용 프로토타입에 적합합니다.

🚀 주요 기능
1. 역운동학(Inverse Kinematics) 경로 생성

목표 좌표 (x, y, z) 입력

각 관절(서보 모터)의 각도 자동 계산

두 지점 사이를 **선형 보간(linear interpolation)**으로 부드럽게 이동

2. 안전 구역(Safe Zone) 검증

하드웨어 보호를 위해 이동 전 좌표를 사전 검사합니다.

뒤쪽 이동 방지: 로봇 기준 뒤편 영역 차단

바닥 충돌 방지: 최소 높이 미만 접근 제한

자기 충돌 방지: 긴 그리퍼 사용 시 몸통과의 충돌 차단

3. 작업 공간(Workspace) 스캔

로봇이 실제로 도달 가능한 모든 좌표를 3D 공간에서 탐색

1cm 단위 전수 조사

X, Y, Z축별 최대 / 최소 도달 범위 출력

프로젝트 설계 시 유효 작업 반경 가이드라인 제공

📂 파일 구성 및 설명
IK.py — Main Control

로봇의 실제 움직임을 제어하는 핵심 스크립트입니다.

주요 함수:

calculate_angles
삼각함수를 이용해 4개 관절의 각도 계산

solve_ik_auto
서보 모터 가동 범위(0~180°) 내에서 최적의 IK 해 자동 탐색

get_linear_path
두 좌표 사이를 직선 경로로 분할하여 부드러운 궤적 생성

SafeZone_Check.py — Diagnostic Tool

좌표 이동 전, 하드웨어 안전성을 점검하는 유틸리티입니다.

검증 항목:

X축 제약: 로봇 등 뒤 영역 (X < 0) 이동 차단

반경 제약:

최소 반경: 15cm (몸통 충돌 방지)

최대 반경: 34cm (물리적 사거리 초과 방지)

높이 제약: 최소 높이 2cm (바닥 충돌 방지)

workspace.py — Range Analyzer

로봇의 물리적 한계를 계산하는 분석 도구입니다.

####기능:

1cm 간격으로 전체 공간 스캔

도달 가능한 좌표만 필터링

X / Y / Z 축별 최대·최소 값 출력

작업 공간 설계 및 시뮬레이션 기준 데이터 제공

💻 설치 및 실행 방법
1. 환경 준비

Raspberry Pi Model B

DOFBOT 공식 라이브러리 Arm_Lib 설치 필요

2. 안전성 테스트
python3 SafeZone_Check.py

3. 역운동학 기반 궤적 제어 실행
python3 IK.py

⚠️ 주의사항

본 라이브러리는 로봇 자체 충돌 방지에 초점을 맞추고 있습니다.
실행 전 주변 환경에 장애물이 없는지 반드시 확인하십시오.

IK.py 실행 시 로봇이 급격하게 움직일 수 있습니다.
전원 인가 후, 로봇의 초기 자세 확보를 권장합니다.

실제 하드웨어 특성(서보 편차, 조립 오차)에 따라 미세한 오차가 발생할 수 있습니다.

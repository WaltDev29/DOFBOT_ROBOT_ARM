📂 DOFBOT-IK-Workspace6축 로봇 팔(DOFBOT) 역운동학 제어 및 작업 공간 분석 라이브러리



이 저장소는 Yahboom DOFBOT의 물리적 규격을 바탕으로 한 역운동학(Inverse Kinematics) 계산, 하드웨어 보호를 위한 안전 영역 검증, 그리고 가동 범위 분석 도구를 포함하고 있습니다.



🚀 주요 기능
역운동학(IK) 경로 생성: 목표 좌표 $(x, y, z)$를 입력받아 각 서보 모터의 각도를 계산하고, 지점 간의 선형 보간 이동을 수행합니다.

안전 구역(Safe Zone) 검증: 로봇의 뒤쪽 이동 방지, 바닥 충돌 방지, 그리고 긴 그리퍼로 인해 발생할 수 있는 몸통 충돌(Self-collision)을 사전에 차단합니다.

작업 공간(Workspace) 스캔: 로봇이 물리적으로 도달 가능한 전체 영역을 3D 스캔하여 사용 가능한 좌표 범위를 출력합니다.



📄 파일 설명

1. IK.py (Main Control)로봇의 움직임을 제어하는 메인 스크립트입니다.
calculate_angles: 삼각함수를 이용해 4개 관절의 각도를 도출합니다.
solve_ik_auto: 서보 모터의 가동 범위(0~180도) 내에서 최적의 해를 자동으로 탐색합니다.
get_linear_path: 두 좌표 사이를 직선으로 이동할 수 있도록 세분화된 궤적을 생성합니다.

2. SafeZone_Check.py (Diagnostic Tool)좌표 이동 전, 하드웨어의 안전을 점검하는 유틸리티입니다.
X축 제약: 로봇 등 뒤($X < 0$) 구역 이동을 차단합니다.
반경 제약: 몸통 충돌 방지를 위한 최소 반경(15cm)과 최대 사거리(34cm)를 검사합니다.
높이 제약: 바닥 충돌을 방지하기 위한 최소 높이(2cm)를 검증합니다.

3. workspace.py (Range Analyzer)로봇이 도달할 수 있는 물리적 한계치를 계산합니다.1cm 간격으로 공간을 전수 조사하여 X, Y, Z축의 최대/최소 범위를 출력합니다.실제 프로젝트 설계 시 로봇의 활동 반경을 설정하는 가이드라인으로 활용됩니다.



💻 설치 및 실행

1. 환경 준비: DOFBOT 라이브러리(Arm_Lib)가 설치된 Raspberry Pi Model B 환경에서 실행하십시오.

2. 안전성 테스트: python3 SafeZone_Check.py

3. 궤적 제어 실행: python IK.py
